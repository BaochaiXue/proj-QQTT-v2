# 2026-05-01 SAM2.1 MAX Environment

## Goal

Create and validate a WSL2 Ubuntu conda environment named `SAM21-max` for SAM2.1 on the RTX 5090 Laptop GPU.

## Requirements

- Python 3.12.
- `torch==2.11.0` with CUDA 13.0 wheel/build.
- Latest compatible `torchvision`.
- Latest `facebookresearch/sam2` main branch.
- SAM2.1 Hiera Large checkpoint.
- SAM2 CUDA extension must compile and import successfully; no silent fallback.
- Validate image predictor.
- Validate video predictor with `vos_optimized=True` / `torch.compile`.
- Run still-object round1 three camera views, 30 frames per view, sequentially, and report average milliseconds per frame.
- Write a verify report under `docs/generated/`.

## Constraints

- Do not install `torch==2.5.1`.
- Do not use CUDA 12.1 / 12.4 / 12.8.
- Do not compile against `/usr/local/cuda` if it points to CUDA 12.x.
- Use SAM2.1 config with SAM2.1 checkpoint; do not mix `sam2_hiera_l.yaml` with `sam2.1_hiera_large.pt`.
- Keep external repos and checkpoints outside this repo.
